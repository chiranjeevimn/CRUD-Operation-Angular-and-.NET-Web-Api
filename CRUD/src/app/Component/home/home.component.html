Welcome to the CRUD Operations Application

This web application demonstrates CRUD functionality using Angular as the frontend, .NET Web API as the backend, and SQL Server for data storage.

What is this application about?

This application allows users to perform basic CRUD operations on data stored in an SQL Server database. The operations are handled through an Angular frontend and a .NET Web API backend.

How does it work?

  • Create: Add new records to the database via the Angular form.
  • Read: Retrieve and display records stored in SQL Server from the backend API.
  • Update: Modify existing records by selecting and editing them in the interface.
  • Delete: Remove records from the database using a delete option in the UI.

Technology Stack

This project is built using the following technologies:

  • Frontend: Angular (for building dynamic applications)
  • Backend: .NET Web API (to handle business logic and data interactions)
  • Database: SQL Server (for reliable and efficient data storage)

Features

  • Responsive User Interface with Angular
  • Secure API communication with JWT authentication
  • SQL Server database integration for storing and retrieving data
  • CRUD operations performed through clean and easy-to-use forms
